Boredomdai 2D (Formerly Dominion 2D) Engine Test for Game I.

Controls

  • Key C - fires currently selected weapon.
  • Key X - cycles between bullets and rockets [if you have the latter!].
  • Key Z - Open/Use/Pushwall activation.
  • Arrow Keys move you around. You fire in the way you were walking or ARE walking.

Current Version Specifications:

  • 24x24 pixel object/gridsquare ratio for higher detail. [Previously implemented at 16x16]
  • Basic button "gui" system from scratch for the title screen + menu.
  • Highly object oriented, with KIActor the basis for all in-game objects for the most part. [GUI is the primary exception]
  • Rise of the Triad type pushwalls, switches and touchplates! EXPLORE
  • Secrets and bonuses for finding them. Kill all enemies for a bonus. Kill no enemies for a bonus.
  • Three levels, lovingly created in Ogmo Editor to try at the moment.
